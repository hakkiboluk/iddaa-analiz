# ⚽ İddaa Analiz Pro v4

Mackolik verilerini kullanan, AI destekli iddaa analiz aracı.

## 🚀 Özellikler
- 📋 Canlı Bülten — Mackolik'ten anlık maç verileri ve oranlar
- 🤖 AI Kupon — 4 farklı risk seviyesinde otomatik kupon önerileri
- 🗄️ Veritabanı — Geçmiş maç sonuçları, takım istatistikleri
- 📜 Kupon Geçmişi — Kazanç/kayıp takibi, otomatik sonuç kontrolü
- 🎯 Risk Analizi — Lig güvenilirliği, oran anomali tespiti
- 🌙/☀️ Tema — Gece/gündüz modu
- ☁️ GitHub Sync — Veriler otomatik senkronize
